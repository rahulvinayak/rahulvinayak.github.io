# 🌐 Personal Portfolio

This repository contains my personal portfolio website showcasing my work in **Electronics, Embedded Systems, and Software Development**.  
It is hosted using **GitHub Pages**.

---

## 🚀 Live Website
🔗 [View Portfolio](https://rahulvinayak.github.io/)  

---

## 📂 Contents
- **index.html** → Main portfolio page  
- **assets/** → Images, icons, and other static files (optional)  
- **style.css** → Custom styling (if separated from HTML)  

---

## 🛠️ Features
- Dark-themed responsive design  
- About Me section  
- Skills & Tools (STM32, ESP32, FreeRTOS, Arduino, KiCad, etc.)  
- Projects with links to GitHub repos  
- Contact section  

---

## 📌 Setup Locally
To run the portfolio on your computer:

```bash
# Clone the repo
git clone https://github.com/rahulvinayak/portfolio.git

# Navigate into folder
cd portfolio

# Open in browser
open index.html   # (or double-click index.html on Windows)
